==========================================
  MASTER SITE DASHBOARD
  VS Code Extensions Reference
==========================================

ESSENTIAL (auto-installed by setup_project.ps1)
-------------------------------------------------
ritwickdey.LiveServer
  - Runs index.html in browser with live reload
  - Right-click index.html > Open with Live Server

esbenp.prettier-vscode
  - Auto-formats HTML / CSS / JS on save
  - Set as default formatter in VS Code settings

dbaeumer.vscode-eslint
  - Catches JavaScript errors as you type
  - Requires no config for basic JS use

ecmel.vscode-html-css
  - CSS class autocomplete inside HTML files

naumovs.color-highlight
  - Shows color swatches next to hex codes in CSS

christian-kohler.path-intellisense
  - Autocompletes file paths in script/link tags

usernamehw.errorlens
  - Shows errors inline on the same line (no hover needed)

eamodio.gitlens
  - Git blame, history, and diff tools inside VS Code

aaron-bond.better-comments
  - Color-codes // TODO // FIXME // ENGINE comments

formulahendry.auto-rename-tag
  - Renames closing HTML tag when you edit the opening tag

------------------------------------------
OPTIONAL (install manually if needed)
------------------------------------------
PKief.material-icon-theme
  - Better file icons (js / css / html / json all distinct)
  cmd: code --install-extension PKief.material-icon-theme

codeium.codeium
  - Free AI code completion (alternative to Copilot)
  cmd: code --install-extension codeium.codeium

firefox-devtools.vscode-firefox-debug
  - Debug JS in Firefox from VS Code
  cmd: code --install-extension firefox-devtools.vscode-firefox-debug

------------------------------------------
NOT NEEDED FOR THIS PROJECT
------------------------------------------
- Python extension     (no backend in this tool)
- Node / npm tools     (no build step required)
- React / Vue / Svelte (plain JS only)
- Docker               (local file-based tool)

==========================================
  HOW TO RUN
==========================================
1. Place setup_project.ps1 and RUN_SETUP.cmd
   in the PARENT folder where you want the
   project created (e.g. C:\Projects\)

2. Double-click RUN_SETUP.cmd

3. A folder "MasterSiteDashboard" is created
   with all engine files and structure ready.

4. VS Code opens automatically.

5. Open index.html, click "Go Live" bottom bar.
